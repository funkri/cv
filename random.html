Sweet & Savoury

When cooking something sweet, always add a small pinch of salt.
Savoury dishes should also have a pinch of sugar.
This adds balance to the flavours of the dishes.

This paragraph contains some words or phrases that are more important than others. These may be emphasized by giving them a class using the span-command with a class identifier. This way I may refer to this class in my CSS-document, and quickly change the way I show emphasis throughout my entire page.

id header

id is used to style specific html elements. It is similar to classes, but only for one element on a page.

HTML bookmarks with id and links

With a long page it might be useful to be able to jump down to a specific section without having to scroll. Create a bookmark by giving this section's heading an id. Then you create a link that refers to this id.
